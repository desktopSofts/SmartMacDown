# SmartMacDown
技术结构
1. vue
2. electron
3. mavon-editor


## 开始
```
npm i
```
另如果要加强调试效果可参考如下方案：

[**Debug环境搭建**](https://blog.csdn.net/lichong951/article/details/126718928)

### 如搭建调试环境可按F5。如没有则输入如下命令
```
npm run electron:serve
```
