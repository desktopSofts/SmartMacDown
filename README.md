## 使用说明
|快捷键|功能说明|备注|
|-|-|-|
|CmdOrCtrl+N|新建md文档||
|CmdOrCtrl + S|保存md文档||
|CmdOrCtrl+F|搜索文件列表||
|CmdOrCtrl+O|导入文件列表||
|CmdOrCtrl + Z|撤销||
|Shift + CmdOrCtrl + Z|重做||
|CmdOrCtrl + X|剪切||
|CmdOrCtrl + C|复制||
|CmdOrCtrl + V|粘贴||
|CmdOrCtrl + R|刷新||
|视图--开发者工具|开发者工具||
|Mac：Ctrl+Cmd+F；win：F11|最大化||
|CmdOrCtrl + M|最小化||
